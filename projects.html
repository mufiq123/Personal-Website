  • Used ChartJS to create a line graph from the data fetched from the API call.
  • Used JavaScript, HTML, and the React framework to build a one-page website that allowed the user to shorten their URLs and redirect them to the page that they want
  • Used Python and Firebase to create the backend process of creating the link that redirects the user and used a NoSQL database to store the links created by the user.
  • Used Swift to develop an IOS weather app that gets the weather of the user's location
  • Used OpenWeather API to use the users current location to get an accurate and detailed weather report
  • Used SwiftUI features to create a UI that would display the information in an easy to understand way
  • Used C++ and Assembly to code a game of pong that can then be displayed on an LCD Screen
  • Used Eagle to create a PCB board that could then get signals from a microcontroller to display the created game of pong
  • Used physical sliders to get the input position of each paddle and updated this on the LCD Screen
  • Used Java to create a backend app that allows a customer to bid on a select number of items in an auction
  • Used a MySQL database to store customer login info, as well as the history of that customers bids and purchase history
  • Used JavaFX to create the frontend which displays the items and the functionality
  • Used Python and Numpy to build upon an algorithm that can calculate the movement of large galactic objects
  • Used this algorithm and MatplotLib to create a visual diagram of the movement and collision of galaxies
  • From this diagram, created a research report on how the initial conditions of galaxies effects the collision that is produced
